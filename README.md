# News App - Real-Time News Discovery Application 🗞️🌍

News App is a sleek and powerful Flutter application that provides users with real-time news from across the globe. Featuring category-based navigation, source filtering, detailed views, and fast search with pagination, it ensures you’re always updated with the latest headlines.

## Features ✨

### 🗂️ Category Navigation

- **7 Main Categories**: Instantly access top news in _General_, _Business_, _Sports_, _Health_, _Science_, _Technology_, and _Entertainment_.
- **Dynamic Routing**: Tapping a category navigates users to a news feed specific to that topic.

### 📰 Sub-Categories (Sources)

- **TabBar for Sources**: View articles from different news sources (e.g., CNN, BBC) inside each category.
- **Horizontal Scrolling**: Easily swipe between sources using a scrollable tab bar.

### 📄 News Details

- **Modal Bottom Sheet**: Tap any news item to open a modal showing the title, description, image, and date.
- **External Navigation**: One-tap button to open and read the full article in a browser.

### 🔍 Search View

- **Search Bar Access**: Tap the search icon in any category screen’s app bar to start searching.
- **Pagination Support**: Efficient infinite scrolling and fetching of news results using page-wise API requests.

### ⚡ Performance & UX

- **Smooth Pagination**: Implements lazy loading and optimal memory use.
- **Responsive UI**: Fully adaptive to Android and iOS screen sizes.

## Technologies Used 🛠️

- **Flutter**: Cross-platform mobile app development.
- **RESTful APIs**: Fetch live news data from reliable third-party news providers.
- **Riverpod**: Modern and scalable state management.
- **MVVM Architecture**: Promotes testable and clean code organization.
- **Repository Pattern**: Clean abstraction of data sources and logic layers.

## 🛠️ Setup Instructions

- **1. Clone the repository**: git clone https://github.com/AhmedTarek-f/news-app
- **2. Navigate into the project directory**: cd news-app
- **3. Install dependencies**: flutter pub get
- **4. Run the app**: flutter run
